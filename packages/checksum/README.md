# @helix/checksum

Add, remove and validate checksums.

## Installation

Install using [npm](https://www.npmjs.org/):
```
npm install @helix/checksum
```

or using [yarn](https://yarnpkg.com/):

``` yarn
yarn add @helix/checksum
```
